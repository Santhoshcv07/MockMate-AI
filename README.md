# 🤖 MockMate-AI

An AI-powered mock interview platform built to help developers practice technical questions and improve their interview skills.

* **Framework:** ![Next JS](https://img.shields.io/badge/Next-black?style=for-the-badge&logo=next.js&logoColor=white) ![React](https://img.shields.io/badge/react-%2320232a.svg?style=for-the-badge&logo=react&logoColor=%2361DAFB)

## ✨ Features

* AI-driven technical interview simulations.
* Secure user authentication.
* Resume upload and parsing capabilities.
* Clean, responsive UI for an optimal practice environment.
